DREAM5 network inference challenge
==================================

This directory contains the input data, gold standards, and true gene names for the four networks of the challenge. In addition, the Matlab scripts used for the prediction assessment are included.

INPUT DATA
Described in the original challenge description on the DREAM website: http://wiki.c2b2.columbia.edu/dream/index.php/D5c4

GOLD STANDARDS
Described in Supplementary Note 3. The gold standard for S. aureus was not used for the scoring in the challenge (see Supplementary Note 7.2).

EVALUATION SCRIPTS
See INSTRUCTIONS.txt file.

NETWORKS
Network1: In silico
Network2: S. aureus (not used for scoring)
Network3: E. coli
Network4: S. cerevisiae
